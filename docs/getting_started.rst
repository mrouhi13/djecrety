Getting started
===============

Available Commands
-------------------

* djecrety

Supported Python versions
-------------------------

* Python 3.6
* Python 3.7
* Python 3.8
* Python 3.9

Supported Django versions
-------------------------

* Django 2.x
* Django 3.0
* Django 3.1
* Django 3.2

Installation
------------

.. code-block:: bash

    $ pip install djecrety

Configuration
-------------

Configure ``INSTALLED_APPS``:

.. code-block:: python

    INSTALLED_APPS = (
        ...
        'djecrety',
        ...
    )
